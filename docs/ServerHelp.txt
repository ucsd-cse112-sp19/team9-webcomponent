How to spin up your own server:
1. Install NPM
2. Install node.js 
   2.a Figure out dependencies
   		a. express
   		b. cookie-parser
   	
   2.b npm install dependencies
3. Copy Related files (msgFetch.js, msgSend.js, chatBox.js, chatStream.js, test.html)
   into a new directory (preferably not in our git space to avoid weird pushes)
4. Find the server code on slack and copy it into a file called server.js

5. run: node server.js
6. You should be able to find it on localhost:8082
